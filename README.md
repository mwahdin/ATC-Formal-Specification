# Air Traffic Control (ATC) System - Formal Specification in VDM-SL

این پروژه یک مدل‌سازی صوری (Formal Specification) برای یک سیستم کنترل ترافیک هوایی (ATC) است که با استفاده از زبان **VDM-SL** طراحی، پیاده‌سازی و اعتبارسنجی شده است. هدف این سیستم، مدیریت امن و هوشمند نشست و برخاست هواپیماها، اولویت‌بندی پروازها بر اساس میزان سوخت و وضعیت اضطراری، و کنترل وضعیت باندهای فرودگاه با توجه به شرایط جوی زنده است.

---

## 🚀 ویژگی‌های کلیدی سیستم (Core Features)

* **مدیریت هوشمند وضعیت باندها:** تخصیص خودکار باندهای آزاد (`<FREE>`) به هواپیماهای در حال فرود و تغییر وضعیت بلافاصله آن‌ها به اشغال‌شده (`<OCCUPIED>`).
* **موتور تصمیم‌گیری برج مراقبت (ATC Brain):** دارای سیستم اولویت‌بندی دو کاناله (صف معمولی و صف اضطراری). پروازهای دارای وضعیت بحرانی همیشه در بالاترین سطح اولویت برای فرود قرار می‌گیرند.
* **محاسبه پویا و هوشمند اولویت‌ها:** تابع وزن‌دهی بر اساس میزان سوخت باقی‌مانده و وضعیت اضطراری هواپیما (امتیازدهی پویا از ۱۰ تا ۱۰۰).
* **لایه ریسک و مدیریت تغییرات جوی:** لغو و متوقف کردن هوشمند تمام فرودها و پروازها در هنگام طوفان شدید (`<STORM>`).
* **تضمین پایداری و امنیت با Invariants:** تعریف خطوط قرمز ریاضی در سیستم (مانند یکتا بودن شماره پروازها در حریم هوایی و همبستگی وضعیت صف‌های اضطراری).

## 🛠 نیازمندی‌ها و ابزار اجرا (Prerequisites)

برای اجرا، تست و شبیه‌سازی این مدل صوری، به ابزارهای زیر نیاز دارید:
1. **Visual Studio Code (VS Code)**
2. افزونه **`VDM VSCode`** (ارائه‌شده توسط تیم Overture)
3. **Java Runtime Environment (JRE)** یا JDK نسخه ۱۱ یا بالاتر (جهت اجرای موتور مفسر VDMJ در پس‌زمینه)

## 💻 نحوه شبیه‌سازی و اجرا در VS Code (Simulation Guide)

1. پوشه پروژه را در VS Code باز کنید.
2. از منوی سمت چپ به بخش **Run and Debug** بروید و دکمه سبز رنگ **Launch VDM** (یا کلید `F5`) را فشار دهید تا مفسر زنده در بخش **Debug Console** باز شود.
3. دستورات سناریوی زیر را به ترتیب در کادر پایین کنسول تایپ کرده و اجرا کنید:

```vdm
-- ۱. فعال‌سازی و انتخاب ماژول فرودگاه به عنوان ماژول پیش‌فرض
default AirTrafficControl

-- ۲. تعریف و ثبت یک باند فرودگاه جدید در حالت آزاد
print AddRunway(mk_Runway(mk_token("R1"), <FREE>, nil))

-- ۳. ثبت یک هواپیما در حال پرواز (با شناسه، شماره پرواز، سوخت، وضعیت، ارتفاع و اولویت اولیه)
print AddAircraft(mk_Aircraft(mk_token("IR655"), mk_token("FL-101"), 100, <AIRBORNE>, 30000, 1))

-- ۴. صادر کردن دستور درخواست فرود و تخصیص خودکار باند آزاد به هواپیما
print RequestLanding(mk_token("IR655"))

-- ۵. بررسی وضعیت موتور تصمیم‌گیری برای انتخاب هواپیمای بعدی صف
print GetNextAircraft()
